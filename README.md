📊 Banking Data Analysis | SQL + Power BI

This project focuses on analyzing banking customer data to uncover insights into loans, deposits, customer behavior, and financial risk. The goal is to support better decision-making through data-driven insights.

🔧 Tech Stack

MySQL (Data Cleaning, EDA, Advanced Queries)
Power BI (Dashboard & Visualization)


📌 Key Highlights

Analyzed 30,000+ customer records with 25+ features including income, loans, deposits, and demographics
Performed data cleaning, transformation, and feature engineering (age groups, wealth score, risk segmentation)
Wrote 30+ SQL queries using joins, subqueries, aggregations, and CASE statements
Built interactive dashboards (3 pages) with 6+ KPIs like Total Loans, Deposits, Business Lending, and Accounts
Applied filters on gender, joining year, banking relationship, and advisor level


📈 Key Insights

Medium-income customers contribute the highest deposits
Business lending forms a major portion of total loans
Customer segmentation reveals patterns across occupation, nationality, and risk levels


🎯 Objective

To transform raw banking data into meaningful insights that improve customer targeting, risk assessment, and financial strategy
